Monobit Test

The Monobit test is a statistical test used to determine the randomness of a binary sequence. It checks the proportion of ones in the sequence to ensure that it is approximately equal to the number of zeros.

If the test passes, it means the sequence has a balanced number of ones and zeros, suggesting randomness. If the test fails, it indicates a bias in the sequence.
