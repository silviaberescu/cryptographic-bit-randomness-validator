Welcome to the Cryptographic Bit Randomness Validator

This tool helps validate the randomness of cryptographic keys using various statistical tests. Randomness is a crucial factor in cryptography, and this tool ensures that your keys are securely generated, helping to maintain the strength of encryption systems.

Key Features

Monobit Test

Assesses the randomness of a sequence by counting the number of ones and zeros in subsequences of M bits.

M-bit Test

Computes if the difference between the frequency of overlapping m-bit long patterns is statistically distinct from one of a true random sequence.

Runs Test

Compares whether the difference between the transition from gaps (sequences of zeros) to blocks (sequences of ones) in the given sequence is statistically relevant compared to the transition within a true random sequence.

Autocorrelation Test

Evaluates if a sequence exhibits periodicity or any predictable patterns.

Serial Test

Examines the randomness by checking pairs of bits in the data sequence.
